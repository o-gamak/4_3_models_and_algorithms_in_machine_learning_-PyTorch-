# Прогнозирование температуры поверхности звезды

**Задача астрономии:** определить абсолютную температуру на поверхности звезды по её наблюдаемым характеристикам.

**Описание данных**

Датасет содержит характеристики 240 изученных звёзд:

- `Temperature (K)` — абсолютная температура поверхности (целевая переменная)
- `Luminosity(L/Lo)` — относительная светимость (относительно Солнца)
- `Radius(R/Ro)` — относительный радиус (относительно Солнца)
- `Absolute magnitude(Mv)` — абсолютная звёздная величина
- `Star color` — цвет звезды (определён по спектральному анализу)
- `Star type` — тип звезды (0–5: от коричневого карлика до гипергиганта)

**Решение:** нейронная сеть на базе PyTorch с качеством **RMSE = 4333 K** на отложенной тестовой выборке.

---

### Главные результаты
- Проведён глубокий исследовательский анализ данных (EDA) с кастомными визуализациями распределений, корреляций (включая Phik) и физических зависимостей
- Сгенерировано и предобработано 14 информативных признаков (OneHot-кодирование цвета и типа, масштабирование числовых)
- Разработана **универсальная архитектура нейронной сети** (PiplineNet) с настраиваемыми слоями, dropout и выбором активаций
- Достигнуто качество **RMSE = 4333 K**, **MAE = 2557 K**, **MAPE = 23.3%** на тесте (прирост ~58 K по RMSE к линейной baseline)
- Реализованы продвинутые инструменты визуализации результатов: интерактивные графики факт/прогноз и детальный анализ ошибок
- Создан полностью воспроизводимый и структурированный пайплайн, готовый к адаптации под другие задачи регрессии

### Результаты моделей

|-------------------------|-----------|----------|
| **Улучшенная нейронная сеть** | **4333 K**| **2557 K**| 
| Baseline-нейронная сеть | 4391 K    | 2533 K   | 

**Выбрана нейронная сеть** — лучший результат по RMSE + способность улавливать нелинейные зависимости

### Ключевые инсайты из анализа
- Температура сильно коррелирует со светимостью, радиусом и абсолютной величиной (в соответствии с законами Вина и Стефана-Больцмана)
- Тип звезды и цвет — мощные категориальные предикторы
- Модель лучше всего предсказывает звёзды средней температуры; на экстремальных объектах (гипергиганты, карлики) ошибки выше из-за редкости классов
- Нейросеть демонстрирует преимущество над традиционными физическими расчётами по точности и удобству автоматизации
